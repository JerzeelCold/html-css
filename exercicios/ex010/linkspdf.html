Podemos também indicar a natureza do link usando o atributo rel, este atributo aceita diversos valores alguns deles são:

  • next: Indica que o link envia para proxima parte do documento atual.
  • prev: Indica que o link leva para a parte anterior do documento atual.
  • author: India que é um link que leva ao site do autor do artigo atual.
  • external: indica que o link leva a um site que não faz parte do site atual.
  • nofollow: indica que é um lik não endossado, como um link pago.
